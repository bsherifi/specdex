# Sample PDFs

This directory holds open-licensed sample PDFs you can drop into Specdex
to try the ingest + scan loop without using proprietary docs.

PDF sources:
- US federal procurement standards (Public Domain — federal works).
- W3C PDF test corpus (Public Domain).
- ASME / SAE freely-distributed standards (verify license on each).

To regenerate them, see `scripts/download-sample-pdfs.sh` (not shipped yet).
